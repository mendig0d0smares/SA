fundamentos de banco de dados

O que é um SGBD?

Um SGBD (Sistema Gerenciador de Banco de Dados) é um software usado para criar, armazenar, organizar, gerenciar e acessar bancos de dados. Ele funciona como uma camada entre o usuário/sistema e os dados, garantindo segurança, integridade e eficiência no acesso às informações.

tipos

SGBD Relacionais (RDBMS)

SGBD Não Relacionais (NoSQL)

dicionario de dados


O que é um dicionário de dados?

Um dicionário de dados é um repositório de informações que descreve todos os elementos de dados de um banco de dados, ele funciona como um manual.

Quais os tipos de dados de um banco de dados?

numericos

INT / INTEGER — número inteiro

SMALLINT — inteiro menor

BIGINT — inteiro maior

DECIMAL / NUMERIC — números com precisão definida (ex.: dinheiro)

FLOAT / REAL / DOUBLE — números com ponto flutuante

texto (strings):

CHAR(n) — tamanho fixo

VARCHAR(n) — tamanho variável

TEXT — textos grandes

CLOB — grandes blocos de texto

booleanos:

BOOLEAN — verdadeiro ou falso (TRUE / FALSE)

Tipos Binários:

BLOB — imagens, PDFs, vídeos, etc.

BINARY / VARBINARY — dados binários de tamanho fixo ou variável

Tipos Especiais (dependem do SGBD):

JSON — documentos JSON (PostgreSQL, MySQL 5.7+)

XML — dados estruturados em XML

UUID — identificadores únicos

GEOMETRY / POINT / GIS — dados geoespaciais

ENUM — lista de valores pré-definidos

que é uma modelagem de banco de dados?

A modelagem de banco de dados é o processo de planejar, organizar e estruturar os dados antes de criá-los em um SGBD.

modelo DER

O DER (Diagrama Entidade-Relacionamento) é um modelo gráfico usado na fase de modelagem para representar:

Entidades (tabelas do banco),

Atributos (campos de cada tabela),

Relacionamentos entre entidades (1:N, 1:1, N:M).

Retângulos → entidades

Elipses → atributos

Losangos → relacionamentos

Cardinalidades → 1, N, etc.

modelo MER

O MER é o modelo teórico criado por Peter Chen que dá origem ao DER. Ou seja:

MER → a teoria/abordagem

DER → o diagrama desenhado baseado nessa teoria

MER = conceito/modelo lógico

DER = representação gráfica do MER
